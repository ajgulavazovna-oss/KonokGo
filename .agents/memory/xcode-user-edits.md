---
name: Xcode user edits
description: Пользователь сам редактирует код в Xcode — не перезаписывать его изменения
---

Пользователь самостоятельно меняет значения (размеры, отступы, шрифты и т.д.) прямо в Xcode.

**Правило:** никогда не возвращать старые значения обратно. Вносить только те изменения, которые явно запрошены.

**Why:** пользователь явно сказал об этом после того, как агент перезаписал его правки размеров баннеров.

**How to apply:** перед редактированием файла — читать текущее состояние файла (не опираться на прошлые версии из истории чата), и менять только запрошенное.
